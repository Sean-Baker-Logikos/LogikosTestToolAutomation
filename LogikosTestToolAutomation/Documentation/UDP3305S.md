# UDP3305S Programmable DC Power Supply

# Example Usage

    from LogikosTestToolAutomation.UDP3305S import UDP3305S
    import time

    psu = UDP3305S()

    # Configure channel 1

    psu.set_mode(UDP3305S.Mode.NORM)
    psu.ch1.set_voltage(12.0)
    psu.ch1.set_current(3.0)

    # Turn on channel 1

    psu.ch1.on()

    # Read channel 1 output values

    time.sleep(2)
    r = psu.ch1.read_all()
    print(f"CH1 - {r[0]}V {r[1]}A {r[2]}W")

# Documentation

### UDP3305S Initilization

    UDP3305S(RID = "")
        Creates a connection to a UDP3305S Programmable DC Power Supply via PyVISA.

        RID : If RID is specified, it will attempt to connect to a UDP3305S with the specified PyVISA identifier.
              If no RID is specified, it will search for a connected UDP3305S and connect to the first one found.

### UDP3305S Methods

    set_mode(mode : UDP3305S.Mode)
    get_mode()
        Sets or queries the mode of the power supply channels 1 and 2, either normal, serial or parallel.

            mode : See UDP3305S.Mode

    on()
        Turn on all outputs

    off()
        Turn off all outputs

    lock()
        Lock keys on instrument panel

    unlock()
        Unlock keys on instrument panel

### UDP3305S Channel Methods

The UDP3305S has three channels
* ch1 : maximum of 33V and 5.2A
* ch2 : maximum of 33V and 5.2A
* ch3 : maximum of 6.2V and 3.2A

Channels 1 and 2 can also be combined into a serial or parallel arrangement to increase the maximum voltage or current.
After setting the mode to either UDP3305S.SER or UDP3305S.PARA, the combined channels can be accessed via the following:
* chSER : maximum of 66V and 5.2A
* chPARA : maximum of 33V and 10.4A

Each channel has the following methods.

    set_voltage(value)
    get_voltage()
        Sets or queries the output voltage

            value : voltage in Volts

    set_current(value)
    get_current()
        Sets or queries the current limit

            value : current limit in Amps

    set_OVP(value, state)
    get_OVP()
        Sets or queries the over voltage protection level

            value : over voltage level in Volts
            state : if False, OVP is turned off (defaults to True)

    set_OCP(value, state)
    get_OCP()
        Sets or queries the over current protection level

            value : over current level in Amps
            state : if False, OCP is turned off (defaults to True)

    read_voltage()
        Reads the output voltage in Volts

    read_current()
        Reads the output current in Amps

    read_power()
        Reads the output power in Watts

    read_all()
        Reads the ouput voltage, current, and power

    on()
        Turn the channel output on

    off()
        Turn the channel output off

### Data Types

    UDP3305S.Mode
        NORM    = 0     # Normal
        SER     = 1     # Channels 1 and 2 in Serial
        PARA    = 2     # Channels 1 and 2 in Parallel
